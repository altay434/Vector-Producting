# Vector-Producting
You can calculate the Vector Products easily with this.

    Example:  1-) 5x + 4y + 7z = Ax + Ay + Az
              
                  Ax = 5  ,  Ay = 4 , Az = 7
         
              2-) 3x + 6y + 9z = Bx + By + Bz
              
                  Bx = 3  ,  By = 6 , Bz = 9
